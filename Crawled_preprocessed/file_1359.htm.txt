bitcoin network mining is in fact a competitive business when a user successfully completes confirmation of a transaction block and that block is added to the block chain the successful user receives a predetermined amount of newly created cryptocurrency as a reward for her efforts in addition to an optional transaction fee paid by the user who initiated the transaction the reward incentivizes the
